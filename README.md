## Fitur Utama
* **Authentication Simulation:** Modul otentikasi sederhana untuk menginisialisasi sesi pengguna (username) yang diparsing melalui URL parameters.
* **Dynamic Dashboard:** Menampilkan representasi visual dari koleksi buku yang dirender secara dinamis dari Controller.
* **Review Management (CRUD Simulation):** * Formulir interaktif untuk menambahkan ulasan baru.
  * Fitur penyuntingan (edit) yang secara otomatis memuat teks dan rating terdahulu.
* **Data Filtering & Array Manipulation:** Logika pemrosesan di backend menggunakan fungsi native PHP (`array_filter`, `array_unshift`) untuk menghindari duplikasi data saat pengguna memperbarui ulasan.
* **Automated Profile Generation:** Pembuatan alamat surel akademik (`@student.unej.ac.id`) secara otomatis berdasarkan input username.
